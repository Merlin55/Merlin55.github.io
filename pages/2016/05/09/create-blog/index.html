一. Binary Search

  • 二分搜索的复杂度是log级别，因此看到follow up复杂度为log时首先考虑二分。
  • 二分搜索使用的前提是有序数组，这里的有序可以使题目直接给定，rotated array，有时还需要自己去构造，例如求最优解。

二分搜索的常用模板

binary search 问题转化成： 寻找第⼀个、最后⼀个、任意一个， 该target元素出现的位置的问题

class Solution:
    # @param {int[]} A an integer array sorted in ascending order
    # @param {int} target an integer
    # @return {int} an integer
    def findPosition(self, A, target):
        # Write your code here
        #首先判断数组是否为空
        if A is None or len(A) == 0:
            return -1
        #起始标记和末尾标记
        start = 0
        end = len(A) - 1

        #start + 1 < end 表示，当指针知道连个元素相邻的时候停止，
        #这样的话在最终分情况讨论的时候， 只⽤考虑 1〜2 个元素。
        while start + 1 < end:
            mid = start + (end - start) / 2 #取中点缩小规模
            if A[mid] == target:            #比较中点的值与目标值的大小
                return mid                  
            elif A[mid] < target:           
                start = mid
            else:
                end = mid
        if A[start] == target:              #最终只剩两个元素，对这两个元素进行讨论
            return start
        if A[end] == target:
            return end
        return -1

需要注意的一点

当比较中点值与目标值大小时，根据任务不同，到底是求target出现的first、last、any位置，进行不同操作

 #求first
 if A[mid] == target: 
     end = mid

 #last
 if A[mid] == target: 
     start = mid

 #any,直接返回
 if A[mid] == target: 
     return mid

First Position of Target [Easy]

For a given sorted array (ascending order) and a target number, find the first index of this number in O(log n) time complexity.

If the target number does not exist in the array, return -1. Example If the array is [1, 2, 3, 3, 4, 5, 10], for given target 3, return 2.

Challenge If the count of numbers is bigger than 2^32, can your code work properly?

class Solution:
    # @param nums: The integer array
    # @param target: Target number to find
    # @return the first position of target in nums, position start from 0 
    def binarySearch(self, nums, target):
        # write your code here
        if nums is None or len(nums) == 0:
            return -1

        start = 0
        end = len(nums) - 1

        while start + 1 < end:
            mid = start + (end - start) / 2
            if nums[mid] < target:
                start = mid
            else:
                end = mid

        if nums[start] == target:
            return start
        if nums[end] == target:
            return end
        return -1
Last Position of Target [Easy]

Find the last position of a target number in a sorted array. Return -1 if target does not exist. Example Given [1, 2, 2, 4, 5, 5]. For target = 2, return 2. For target = 5, return 5. For target = 6, return -1.

class Solution:
    # @param {int[]} A an integer array sorted in ascending order
    # @param {int} target an integer
    # @return {int} an integer
    def lastPosition(self, A, target):
        # Write your code here
        if A is None or len(A) == 0:
            return -1
        start = 0
        end = len(A) - 1
        while start + 1 < end:
            mid = start + (end - start) / 2
            if A[mid] == target:
                start = mid
            elif A[mid] < target:
                start = mid
            else:
                end = mid
        if A[end] == target:
            return end
        elif A[start] == target:
            return start

        return -1

First Bad Version [Medium]

The code base version is an integer start from 1 to n. One day, someone committed a bad version in the code case, so it caused this version and the following versions are all failed in the unit tests. Find the first bad version. You can call isBadVersion to help you determine which version is the first bad one. The details interface can be found in the code's annotation part. Notice Please read the annotation in code area to get the correct way to call isBadVersion in different language. For example, Java is SVNRepo.isBadVersion(v) Example Given n = 5: isBadVersion(3) -> false isBadVersion(5) -> true isBadVersion(4) -> true Here we are 100% sure that the 4th version is the first bad version. Challenge You should call isBadVersion as few as possible.

tips

find first position target 判断该位置是否为bad version

#class SVNRepo:
#    @classmethod
#    def isBadVersion(cls, id)
#        # Run unit tests to check whether verison `id` is a bad version
#        # return true if unit tests passed else false.
# You can use SVNRepo.isBadVersion(10) to check whether version 10 is a 
# bad version.

class Solution:
    """
    @param n: An integers.
    @return: An integer which is the first bad version.
    """
    def findFirstBadVersion(self, n):
        # write your code here
        isBad = SVNRepo()
        start = 1
        end = n
        while start + 1 < end:
            mid = start + (end - start) / 2
            if isBad.isBadVersion(mid):
                end = mid
            else:
                start = mid

        if isBad.isBadVersion(start):
            return start
        if isBad.isBadVersion(end):
            return end

Search for a Range [Medium]

Given a sorted array of n integers, find the starting and ending position of a given target value. If the target is not found in the array, return [-1, -1]. Example Given [5, 7, 7, 8, 8, 10] and target value 8, return [3, 4]. Challenge O(log n) time.

class Solution:
    """
    @param A : a list of integers
    @param target : an integer to be searched
    @return : a list of length 2, [index1, index2]
    """
    def searchRange(self, A, target):
        # write your code here
        #find first position
        if A is None or len(A) == 0:
            return [-1 ,-1]
        position = [-1, -1]    
        start, end = 0, len(A) - 1
        while start + 1 < end:
            mid = start + (end - start) / 2
            if A[mid] == target:
                end = mid
            elif A[mid] > target:
                end = mid
            else:
                start = mid
        if A[start] == target:
            position[0] = start
        elif A[end] == target:
            position[0] = end
        else:
            return position
         #find last position   
        start, end = 0, len(A) - 1
        while start + 1 < end:
            mid = start + (end - start) / 2
            if A[mid] == target:
                start = mid
            elif A[mid] > target:
                end = mid
            else:
                start = mid
        if A[end] == target:
            position[1] = end
        elif A[start] == target:
            position[1] = start

        return position
Total Occurrence of Target [Easy]

Given a target number and an integer array sorted in ascending order. Find the total number of occurrences of target in the array. Example Given [1, 3, 3, 4, 5] and target = 3, return 2. Given [2, 2, 3, 4, 6] and target = 4, return 1. Given [1, 2, 3, 4, 5] and target = 6, return 0. Challenge Time complexity in O(logn)

tips

  • 最朴素的方式是从左向右遍历，当遇到目标值开始计数，一直到不等于目标值结束。
  • 用二分的思想来做的话，由于是有序数组，那么所有相等的值是放在一起的，那么可以分别查找first、last position，数组下标相减加一便是目标值出现的次数
class Solution:
    # @param {int[]} A an integer array sorted in ascending order
    # @param {int} target an integer
    # @return {int} an integer
    def totalOccurrence(self, A, target):
        # Write your code here
        if A is None or len(A) == 0:
            return 0
        #find first position
        start = 0
        end = len(A) - 1
        while start + 1 < end:
            mid = start + (end - start) / 2
            if A[mid] == target:
                end = mid
            elif A[mid] < target:
                start = mid
            else:
                end = mid
        if A[start] == target:
            first = start
        elif A[end] == target:
            first = end
        else:
            return 0

        #find last position
        start = 0
        end = len(A) - 1
        while start + 1 < end:
            mid = start + (end - start) / 2
            if A[mid] == target:
                start = mid
            elif A[mid] < target:
                start = mid
            else:
                end = mid
        if A[end] == target:
            last = end
        elif A[start] == target:
            last = start

        #下标相减加一得到出现次数   
        return last - first + 1

3.矩阵搜索

关于矩阵，可以用二维数组来表示，一个mn的矩阵matrix，取[x, y]位置的值，matrix[x][y]; 同时矩阵也可以看做是一个长度为mn的一维数组，某一元素在一位数组的中的位置为z，那么在一位数组和二维数组中该元素下角标的对应关系为：

$$ x = z / n $$
$$ y = z\%n $$

Search a 2D Matrix [Easy]

Write an efficient algorithm that searches for a value in an m x n matrix. This matrix has the following properties: Integers in each row are sorted from left to right. The first integer of each row is greater than the last integer of the previous row. Example Consider the following matrix: [ [1, 3, 5, 7], [10, 11, 16, 20], [23, 30, 34, 50] ] Given target = 3, return true. Challenge O(log(n) + log(m)) time

Search a 2D Matrix II [Medium]

Write an efficient algorithm that searches for a value in an m x n matrix, return the occurrence of it. This matrix has the following properties: Integers in each row are sorted from left to right. Integers in each column are sorted from up to bottom. * No duplicate integers in each row or column. Example Consider the following matrix: [ [1, 3, 5, 7], [2, 4, 7, 8], [3, 5, 9, 10] ] Given target = 3, return 2. Challenge O(m+n) time and O(1) extra space

tips

  • 与上一题相比，不同之处在于，只保证每一行是有序的，但是每一行头对尾变为一位数组后不是有序的，类似于ratated array。并且题目要求求出现次数，所以不能使用一次二分就能解决。除非是分别对每一行进行二分然后再累加起来，那么此时的复杂度是O(mlogn)。
  • 对于矩阵类的题目，走方格等等之类的题目，从一个角走到对角要走m+n步。对于本题，如果从左上角走到右下角，只能向下或向右走(不能往回走)，下面的和右面的都比自己大，此时就无法辨别到底怎么走。若从右上向左下走，下面的比自己大，右边的永远比自己小，那么就可以根据比较自己与目标元素的大小，来决定是向下走还是向右走。若target比当前元素小，那么应该向右走；若target比当前元素大，那么应该向下走；若target与当前元素相等，因为左小和下大，唯一的可能只能沿着对角线向左下走一步的因最多为要走m+n步，所以复杂度是O(m+n)。
class Solution:
    """
    @param matrix: An list of lists of integers
    @param target: An integer you want to search in matrix
    @return: An integer indicates the total occurrence of target in the given matrix
    """
    def searchMatrix(self, matrix, target):
        # write your code here
        if matrix is None or len(matrix) == 0 or matrix[0] is None or len(matrix[0]) == 0:
            return 0
        m, n = len(matrix) - 1, len(matrix[0]) - 1
        x, y = 0, n
        num = 0
        #循环条件，x，y不能越界
        while x <= m and y >= 0:
            if matrix[x][y] == target:
                num += 1
                x +=1
                y -= 1
            elif matrix[x][y] > target:
                y -= 1
            else:
                x += 1

        return num
